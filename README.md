# Arch Linux Installation Scripts

1. Boot up PC using an Arch Linux Installation Medium
2. Connect to the internet using ethernet or iwd
3. Install git by running `pacman -Sy && pacman -S git`
4. Clone this repo by running `git clone https://github.com/ethmth/install-scripts.git /root/install-scripts`
5. Run `/root/install-scripts/1-arch-install-format.sh` and follow the instructions.
